Informații generale

HyperText Markup Language (HTML) este un limbaj de marcare utilizat pentru crearea paginilor web ce pot fi afișate într-un browser (sau navigator). Scopul HTML este mai degrabă prezentarea informațiilor – paragrafe, fonturi, tabele ș.a.m.d. – decât descrierea semanticii documentului. În cadrul dezvoltării web de tip front-end, HTML este utilizat împreună cu CSS și JavaScript

Specificațiile HTML sunt dictate de World Wide Web Consortium (W3C), care încurajează utilizarea CSS în locul a HTML explicit pentru prezentare.

Noțiuni de bază

HTML este prescurtarea de la Hyper Text Mark-up Language si este codul care sta la baza paginilor web.

Paginile HTML sunt formate din etichete sau tag-uri și au extensia „.html” sau „.htm”. În marea lor majoritate aceste etichete sunt pereche, una de deschidere <eticheta> și alta de închidere </eticheta>, mai există și cazuri în care nu se închid, atunci se folosește <eticheta />. Navigatorul web interpretează aceste etichete afișând rezultatul pe ecran. HTML-ul este un limbaj care nu face deosebire între litere majuscule și minuscule.

Pagina principala a unui domeniu este fisierul „index.html” respectiv „index.htm”. Această pagină este setată a fi afișată automat la vizitarea unui domeniu.

De exemplu la vizitarea domeniului www.nume.ro este afișată pagina www.nume.ro/index.html.

Unele etichete permit utilizarea de atribute care pot avea anumite valori: 

<eticheta atribut="valoare"> ... </eticheta>

Componența unui document HTML este:

  1. versiunea HTML a documentului
  2. zona head cu etichetele <head> </head>
  3. zona body cu etichetele <body> </body> sau <frameset> </frameset>

Versiunea HTML poate fi:

  • HTML 4.01 Strict 
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
  • HTML 4.01 Transitional 
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
  • HTML 4.01 Frameset 
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
  • HTML 5 
<!DOCTYPE HTML>

Toate paginile HTML încep și se termină cu etichetele <html> și </html>.

În interiorul acestor etichete găsim perechile <head>, </head> și <body>, </body>.

head conține titlul paginii între etichetele <title> și </title>, descrieri de tip <meta>, stiluri pentru formatarea textului, script-uri și legături către fisiere externe (de exemplu script-uri, fișiere de tip CSS sau favicon).

Etichetele de tip meta conțin cuvinte cheie, descrierea paginii, date despre autor, informații utile motoarelor de căutare și au următorul format: 

<META NAME="nume" CONTENT="continut">

Exemplu: link către un fisier extern CSS: 

<link rel="stylesheet" type="text/css" href="css.css">

body găzduiește practic toate etichetele afișate de browser pe ecran.

Exemplu: o pagină HTML cu titlul Exemplu iar conținutul Conținut pagină 

<html>
    <head>
    <title>Exemplu</title>
    </head>
    <body>
    Conținut pagină
    </body>
    </html>

Și în HTML poate fi introdus un comentariu, care bineînțeles nu va fi afișat de browser: 

<!-- comentariu -->

Istoria HTML-ului

HTML a fost creat pentru prima dată de Tim Berners-Lee, Robert Cailliau și alții începând cu 1989. Înseamnă HyperText Markup Language.

Hipertext înseamnă că documentul conține legături care permit cititorului să sară în alte locuri din document sau într-un alt document. Cea mai recentă versiune este cunoscută sub numele de HTML5.

Un limbaj de marcare este un mod prin care computerele vorbesc între ele pentru a controla modul în care textul este procesat și prezentat. Pentru a face acest lucru, HTML folosește două lucruri: etichete și atribute.

Ce sunt etichetele și atributele?

Etichetele și atributele sunt baza HTML.
Ele lucrează împreună, dar îndeplinesc funcții diferite - merită să investești 2 minute în diferențierea celor două.

Etichetele sunt folosite pentru a marca începutul unui element HTML și sunt de obicei închise între paranteze unghiulare. Un exemplu de etichetă este: <h1>.

Majoritatea etichetelor trebuie să fie deschise <h1> și închise </h1>. pentru a funcționa.

Atributele conțin informații suplimentare. Atributele iau forma unei etichete de deschidere și informații suplimentare sunt plasate în interior.

Un exemplu de atribut este:

<img src="mydog.jpg" alt="A photo of my dog.">

În acest caz, sursa de imagine (src) și textul alternativ (alt) sunt atribute ale etichetei <img>.

Editorii HTML

În primul rând, trebuie să ne asigurăm că avem instrumentele potrivite. Cel mai important, avem nevoie de un editor HTML.

Există multe opțiuni pe piață. Iată câteva dintre cele mai populare:

Sublime Text 3

Pros

  • Ușor de personalizat
  • Pentru începători
  • Scheme de culori plăcute din care să alegeți.

Cons

  • Nu se pot imprima documente sau coduri
  • Nu este disponibilă nicio bară de instrumente sau tablou de bord.

Notepad ++

O altă alegere obișnuită pentru coderii HTML și alte limbaje este Notepad ++. Este un mic program pentru a descărca și a efectua funcțiile de care aveți nevoie pentru scrierea unui cod curat.

Pros

  • Interfață fără distragere a atenției
  • Caracteristică de completare automată
  • Opțiuni de plugin pentru funcționalități extinse.

Cons

  • Poate fi dificil să te obișnuiești pentru începători
  • Nu există suport pentru Mac.

Etichete suplimentare

În interiorul etichetei <head>, există o etichetă care este întotdeauna inclusă: <title >, dar există altele la fel de importante:

<title>
Aici inserăm numele paginii, așa cum va apărea în partea de sus a ferestrei browserului sau a filei.

<meta>
Aici sunt stocate informații despre document: codarea caracterelor, numele (contextul paginii), descrierea.

        <title>My First Webpage</title>
        <meta charset="UTF-8">
        <meta name="description" content="This field contains information about your page. It is usually around two sentences long.">.
        <meta name="author" content="Conor Sheils">
